# Introdução à Análise de Dados com Python

Repositório contendo o material utilizado no minicurso realizado na UERN.

## Projeto Física em Diálogo

O minicurso faz parte do projeto **Física em Diálogo**, uma iniciativa do Departamento de Física da Universidade do Estado do Rio Grande do Norte (UERN), coordenada pela Professora Dra. Maria Aldinez Dantas.

O projeto possui três eixos principais:

- **Ponto Triplo** — ciclo de palestras;
- **Minicursos** — atividades de formação e capacitação;
- **CineCiências** — exibição e discussão de filmes relacionados à ciência.

Este minicurso está inserido no subeixo **Codigamente**, vinculado ao eixo de minicursos, com foco na introdução à programação, análise de dados e ferramentas computacionais aplicadas à ciência.

## Resumo

O objetivo deste minicurso é apresentar os princípios básicos da análise de dados em Python, concentrando-se nas bibliotecas `pandas` e `matplotlib`. Ao longo de quatro encontros, os participantes aprenderão técnicas básicas de manipulação, organização e visualização de dados utilizando dados da compilação Pantheon+.

A proposta é apresentar uma abordagem prática e acessível, permitindo que os alunos desenvolvam familiaridade com ferramentas amplamente utilizadas em análise de dados e pesquisa científica. Ao término do curso, os participantes estarão aptos a executar análises simples, explorar conjuntos de dados e criar gráficos básicos utilizando Python.

Recomenda-se utilizar:

- Python 3.10+
- Jupyter Notebook

- Matplotlib

- Pandas

## Instalação

Clone o repositório:

```bash
git clone git@github.com:zouzaxd/minicurso-UERN.git
```

Entre na pasta:

```bash
cd minicurso-UERN
```

## Executando os notebooks

Os notebooks podem ser executados localmente utilizando o Jupyter Notebook/JupyterLab ou diretamente no Google Colab.

### Opção 1 — Jupyter Notebook (local)

Abra o Jupyter Notebook:

```bash
jupyter notebook
```

ou:

```bash
jupyter lab
```

### Opção 2 — Google Colab

Os notebooks também podem ser abertos diretamente no Google Colab, permitindo a execução online sem necessidade de instalação local do Python.

Basta acessar:

- https://colab.research.google.com/

E fazer upload do notebook `.ipynb` desejado.
